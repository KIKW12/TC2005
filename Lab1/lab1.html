Preguntas Laboratorio 1

¿Cuál es la diferencia entre Internet y la World Wide Web?

Internet es una red de computadoras interconectadas alrededor del mundo, mientras que la WWW es una enorme coleccion de paginas que se encuentra en Internet.

¿Cuáles son las partes de un URL?

  • Protocolo (http://, https://)
  • Subdiminio (www., contacto.)
  • Nombre de dominio (google.com, tec.mx)
  • Extension (.es, .com)
  • Ruta (/contacto, /blog)
  • Parametros (?q=.....)

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

  • GET: Se usa para obtener datos de un servidor
  • HEAD: Se usa para obtener la cabecera de una respuesta HTTP
  • POST: Se usa para crear un nuevo recurso
  • PUT: Se usa para modificar un recurso existente con una version actualizada
  • PATCH: Se usa para actualizar un recurso existente
  • DELETE: Se utiliza para eliminar un recurso de la base de datos

¿Qué método HTTP se debe utilizar al enviar un formulario HTML?

Se utiliza el método POST

¿Qué método HTTP se utiliza al acceder a una página mediante URL?

Se utiliza el método GET

¿Qué significa un código 200?

Que la solicitud se hizo de manera correcta

¿Es responsabilidad del desarrollador corregir un error 404?

Sí, ya que siginifica que un error de ruta se produjo, causado por errores de enlace, errores de redirección, etc.

¿Es responsabilidad del desarrollador corregir un error 500?

Sí, ya que es un error interno del servidor

¿Qué significa que un atributo HTML5 esté depreciado?

Que los navegadores aun la admiten pero ya no forma parte de la especificacion HTML oficial

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?

  • HTML5: admite audio, video, graficos, tiene controles de formulatios avanzados, mejor compatibilidad entre navegadores
  • HTML4: depende de complementos de terceros para elementos graficos como Flash Player, tiene elementos basicos de formularios, no es tan compatible entre navegadores

¿Qué componentes tiene una tabla?

  • < table >: Etiqueta contenedora de la tabla
  • < tr >: Fila de la tabla
  • < th >: Celda de encabezado
  • < td >: Celda de datos

¿Cuáles son los principales controles de una forma HTML5?

  • text, password, email, number, tel
  • radio, checkbox
  • select, textarea
  • date, time, datetime-local
  • file, color
  • range, search
  • submit, reset, button

¿Cuál es el ciclo de vida de los sistemas de información?

Enfoque por fases de análisis y diseño que anilizan que los sistemas son desarrollados por un ciclo de actividades del analista y del usuario

¿Cuál es el ciclo de desarrollo de sistemas de información?

Mapa que se usa como guía a través de cada etapa del desarrollo, permitiendo la planificación, el seguimiento y la evaluación de los sistemas
